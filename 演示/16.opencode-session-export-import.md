参考这个目录的 说明

1. 先查看有哪些 session                                                                                      
                                                                                                                                             
     opencode session list                                                                                                            
                                                                                                                                
     会列出所有 session，包含 ID 和标题。                                                                                            
                                                                                                                              
     2. 导出单个 session                                                                                                                     
                                                                                                                                       
     # 导出到文件（避免状态消息混入 JSON）                                                                   
     opencode export <session_id> 2>/dev/null > backup.json                                                              
                                                                                                                         
     3. 完整示例                                                                                                     
                                                                                                                                             
     # A 机器上操作                                                                                                                 
     opencode session list                                                                                        
     # 输出：                                                                                                                     
     # ses_abc123  修复登录 bug          2026-03-19                                                                                
     # ses_def456  添加用户模块          2026-03-18                                                             
     # ses_ghi789  重构数据库层           2026-03-17                                                                                         
                                                                                                                                             
     # 导出想要的那个                                                                                                                        
     opencode export ses_abc123 2>/dev/null > login_bug_session.json                                                                         
                                                                                                                                             
     # 复制到 B 机器                                                                                                                         
     # (用 U盘/网盘/微信/邮件...都可以)                                                                                                      
                                                                                                                                             
     4. B 机器上导入                                                                                                                      
                                                                                                                                             
     # 进入你想关联的项目目录                                   
     # 进入你想关联的项目目录                                                                                                        
     cd ~/my-project                                                                                                 
                                                                                                                              
     # 导入                                                                                                                    
     opencode import login_bug_session.json                                                                   
                                                                                                                                             
     导入成功后会有提示：                                                                                                                    
                                                                                                                                             
     Imported session: ses_abc123                                                                                                            
                                                                                                                                             
     然后用 /sessions 就能看到导入的 session 了。
